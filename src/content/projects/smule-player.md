A personal project built to explore playlist management and audio streaming in a web context.

## Background

The Smule app doesn't offer a way to browse a singer's full performance history in one place. This project pulls together recordings into a browsable, searchable player interface.

## Technical highlights

- Angular component architecture with lazy-loaded routes
- Integration with the Smule public API
- Custom audio player controls with progress tracking
- Responsive layout for mobile and desktop

## What I learned

Working with third-party media APIs — rate limiting, auth, and the quirks of streaming embedded audio — plus building accessible media controls from scratch.
